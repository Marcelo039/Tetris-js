🎮 Tetris Dev — Versionamento  com Git

Construção de um jogo Tetris com foco em evolução incremental, boas práticas de versionamento e fluxo profissional com Git.

📌 Sobre o Projeto

Este projeto tem dois objetivos principais:

Desenvolver um jogo clássico estilo Tetris utilizando tecnologias web.

Aplicar e fixar fluxo profissional de Git, simulando ambiente real de desenvolvimento.

Mais do que um jogo, este repositório representa um laboratório prático de:

Organização de código

Estruturação de branches

Commits semânticos

Resolução de conflitos

Evolução incremental de software
